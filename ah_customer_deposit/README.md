Pos Margin Sale
## [Pos Margin sale](index.html)
This module calculates and enforces the minimum sale price and margin for each product. It triggers a warning if a product is sold below its minimum price in sale or POS orders, helping maintain profitability

## Features
- **Warning Message For Product Under Minimum Sales Price on Module Sales Order**
It will automatically give you a warning if an item is sold below the specified margin standard

- **Setup Blocking the next process when selling below the minimum price**
It will automatically stop the process of going to the next page if a sale price is below the minimum price

- **Setup Margin product bsae on category**
You can set product margins based on categories without having to open products one by one

## Installation
1. Download the module from this repository.
2. Install the module in your Odoo instance.
3. Install module Point Of sale and module sales order, After this you can install addons pos margin sale.

## Usage
 After installation, you can configure the settings then check or uncheck the "Blocking Transaction Order" configuration.

## Compatibility
Odoo version: 17.0
