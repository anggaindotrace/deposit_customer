Deposit Customer

This module will help you to manage customer deposits, so that you will not have difficulty if there are payments using the deposit model in your business

Overview

    After installing, you must setup Accounting -> Journal and make sure the customer deposit journal is available

    Don't forget to register the customer deposit journal with the POS payment method too, at POS -> Configuration -> Payment Method

    After that, you can set policies in the settings regarding permission to continue transactions or not being able to continue when the selling price is below the minimum price. Settings in Sales Order and POS Order Checklist if you do not want to continue the transaction and uncheck list if you want to continue the transaction

    POS ORDER CONFIG

    SALES ORDER CONFIG

    You can also make group margin changes using this method

    SETUP MARGIN BASE ON CATEGORY PRODUCT

Benefits

  • Profitability: Ensures products are sold above their minimum price, safeguarding profit margins.
  • Efficiency: Automatically warns if products are underpriced, reducing manual oversight.
  • Informed Decisions: Supports sales teams in focusing on profitable deals.

Installation

  1. Download the module from this repository.
  2. Install the module in your Odoo instance.

Compatibility

Compatible with Odoo version: 17.0
